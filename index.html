Personal Space Invaders: An arcade style shooter

Demo Github Code

The goal of this project was to create a simple game with elements created in Javascript or Jquery. The starting point for this game was creating the basic grid layout with a list of 'divs' in the html. The aliens and lasers were created by applying classes to the elements within the grid.

Whack a mole game: A grid based game using Javascript and Jquery

Demo Github code

This project utilised the set Interval and clear Interval command in Javascript. The mole in the game is situated on a grid and would appear randomly when the start button was clicked.
